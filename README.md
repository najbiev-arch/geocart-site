# Локальная версия сайта Geocart

Статическая версия сайта: `index.html`, `styles.css`, `app.js`, страницы сервисов
`urban-calculator.html` и `topology-check.html`.

## Как открыть

Откройте `index.html` в браузере.

## Языки и тема

Переключение RU / KZ / EN и темная / светлая тема находятся в шапке сайта.
Тексты интерфейса, карточек услуг и проектов задаются в словарях `I18N`, `SERVICES`
и `PROJECTS` в файле `app.js`.

## Услуги

Блок "Продукты и услуги" рендерится из массива `SERVICES` в `app.js`.
Ряд карточек горизонтально прокручивается, сейчас в нем 5 направлений: GIS, URB,
UAV, AGR и EDU.

## Карта проектов

Проекты лежат в массиве `PROJECTS` в `app.js`. Для каждой точки нужны:

- `title`, `place`, `summary` - тексты на `ru`, `kk`, `en`
- `year` - год
- `category` - код категории для фильтра
- `lat` и `lng` - координаты

Карта работает через MapLibre GL JS, использует кластеризацию маркеров и три
подложки: OpenFreeMap Liberty, OpenFreeMap Fiord и Esri World Imagery.

## Контакты

Ссылка кнопки Telegram находится в `index.html`. Сейчас там стоит нейтральная
заготовка `https://t.me/`; замените ее на реальную ссылку компании.

## Публикация через GitHub и Vercel

1. Создайте репозиторий на GitHub, например `geocart-site`.
2. Загрузите в него все файлы из этой папки.
3. В Vercel выберите `Add New Project` и импортируйте репозиторий GitHub.
4. Для статического сайта build-команда не нужна. Vercel сам опубликует `index.html`.
5. После публикации подключите домен в разделе `Project Settings -> Domains`.
6. В панели домена настройте DNS-записи, которые покажет Vercel.

Файл `vercel.json` включает красивые URL без `.html`.
